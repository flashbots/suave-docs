# Community Directory

This page is not a tutorial: you have reached the point in this section where you are ready to branch out into your own work.

Using the key primitives we have walked through together - the onchain-offchain programming model, the confidential store, and external calls (from what will be trusted code running in a TEE) - you can build all kinds of applications on SUAVE.

What smart contract framework and frontend tools you use to ship your work to a larger audience is a matter of personal taste.
