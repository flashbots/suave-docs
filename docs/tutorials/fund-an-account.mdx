We are going to write a script that funds a local account of your choosing, using the Golang SDK.

:::info 

You need to be [running SUAVE locally](/tutorials/run-suave) in order for this script to function. 

:::

Inside your `suave-geth` repo, create a new file in `suave/devenv/cmd` called `fund.go`:

```go
package main

import (
	"context"
	"crypto/ecdsa"
	"fmt"
	"math/big"
	"os"

	_ "embed"

	"github.com/ethereum/go-ethereum/common"
	"github.com/ethereum/go-ethereum/core/types"
	"github.com/ethereum/go-ethereum/crypto"
	"github.com/ethereum/go-ethereum/rpc"
	"github.com/ethereum/go-ethereum/suave/sdk"
)

var (
	// This is the default Kettle address for local development
	exNodeEthAddr = common.HexToAddress("b5feafbdd752ad52afb7e1bd2e40432a485bbb7f")
	exNodeNetAddr = "http://localhost:8545"
	// This is the default funded local development account
	// address: 0xBE69d72ca5f88aCba033a063dF5DBe43a4148De0
	fundedAccount = newPrivKeyFromHex("91ab9a7e53c220e6210460b65a7a3bb2ca181412a8a7b43ff336b3df1737ce12")
)

func main() {
	rpcClient, _ := rpc.Dial(exNodeNetAddr)
	// Use the SDK to create a new client by specifying the Eth Address of the MEVM
	mevmClt := sdk.NewClient(rpcClient, fundedAccount.priv, exNodeEthAddr)

	fundBalance := big.NewInt(100000000)
	if err := fundAccount(mevmClt, common.HexToAddress("<YOUR_ADDRESS_WITHOUT_0x>"), fundBalance); err != nil {
		fmt.Fprintf(os.Stderr, "Error: %v\n", err)
		return
	}
	fmt.Printf("Funded test account: %s (%s)\n", common.HexToAddress("<YOUR_ADDRESS_WITHOUT_0x>"), fundBalance.String())
}

func fundAccount(clt *sdk.Client, to common.Address, value *big.Int) error {
	txn := &types.LegacyTx{
		Value: value,
		To:    &to,
	}
	result, err := clt.SendTransaction(txn)
	if err != nil {
		return err
	}
	_, err = result.Wait()
	if err != nil {
		return err
	}
	// check balance
	balance, err := clt.RPC().BalanceAt(context.Background(), to, nil)
	if err != nil {
		return err
	}
	if balance.Cmp(value) != 0 {
		return fmt.Errorf("failed to fund account")
	}
	return nil
}

// General types and methods we need for the above to work as we want it to,
// nothing SUAVE specific
type privKey struct {
	priv *ecdsa.PrivateKey
}

func newPrivKeyFromHex(hex string) *privKey {
	key, err := crypto.HexToECDSA(hex)
	if err != nil {
		panic(fmt.Sprintf("failed to parse private key: %v", err))
	}
	return &privKey{priv: key}
}
```

Replace `<YOUR_ADDRESS_WITHOUT_0x>` with the address you want to fund.

You can now fund the account you want to by running the following command:

```bash
go run suave/devenv/cmd/fund.go
```

The basic flow of the above script is:

1. Instantiate a client using the Golang SDK. This requires setting up both the RPC and then the "MEVM". If you go back to the [first tutorial](/tutorials/run-suave), you'll see that the MEVM is the part of any Kettle which is responsible for executing any required computation.
2. Specify an amount to fund your account with.
3. Send a transaction funding the account you specify (make sure to leave off the "0x").

:::
