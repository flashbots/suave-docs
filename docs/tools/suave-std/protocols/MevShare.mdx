# Mev-Share

Mev-Share is a library with utilities to interact with the Flashbots Mev-Share API described in https://github.com/flashbots/mev-share/blob/main/specs/bundles/v0.1.md#json-rpc-request-scheme.

## Functions

### [sendBundle](https://github.com/flashbots/suave-std/tree/main/src/protocols/MevShare.sol#L77)

Send a Mev-Share to the Flashbots Mev-Share API.

Input:

- `url` (`string`): The URL of the Flashbots Mev-Share API.

- `bundle` (`Bundle`): The Mev-Share bundle to send.
