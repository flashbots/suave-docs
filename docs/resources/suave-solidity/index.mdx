SUAVE solidity gives you: 

- a way to define private compute
- a way to store private data
- a way to build blocks
- a way to interface with other MEV components:
    - existing MEV infra i.e sending bundles to block builders or relays or valdiators or sequencers
    - new MEV infra aka other SUAPPs on SUAVE
