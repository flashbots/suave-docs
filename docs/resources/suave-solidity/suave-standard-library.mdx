A collection of Solidity contracts and libraries to create Suapps. The idea is for this repo to be the main entrypoint for developers to interact with Suave, a place to create building blocks, sdks and patterns that Suave developers might find useful.

Some of the core functionalities of the library handle:
- transaction parsing and construction
- JSON object decoding and encoding
- convenience wrapper around precompiles

## Libraries

### Transactions.sol

Helper library that defines types and encoding/decoding methods for the Ethereum transaction types.

#### Example usage

```solidity
import "suave-std/Transactions.sol";

contract Example {
    function example() {
        Transactions.Legacy memory legacyTxn0 = Transactions.Legacy({
            to: address(0x095E7BAea6a6c7c4c2DfeB977eFac326aF552d87),
            gas: 50000,
            gasPrice: 10,
            value: 10,
            ...
        });

        // Encode to RLP
        bytes memory rlp = Transactions.encodeRLP(legacyTxn0);

        // Decode from RLP
        Transactions.Legacy memory legacyTxn1 = Transactions.decodeRLP(rlp);
    }
}
```
